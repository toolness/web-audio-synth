adventures in audio synthesis

This is my attempt to learn about audio synthesis by generating waveforms in the browser.

This experiment uses ES2015 without a transpiler, so it probably won't work unless your browser is really awesome.

At the time of this writing, that means you need to be using the latest version of Chrome, Edge, or Firefox.

For more details, check out the GitHub repository.
