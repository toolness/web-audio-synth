MIDI Synthesizer

This is an attempt to build a simple MIDI synthesizer. You'll need a MIDI input device and a browser supporting the Web MIDI API to use it.

Current instrument:

Your MIDI device must send a Program Change event to adjust the current instrument.

Pulse wave duty cycle:

Your MIDI device must send a Control Change event for controller 1 to change the pulse wave duty cycle.

Play notes on your MIDI device to make music!
